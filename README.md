# 🌐 Justin Okland – Project Management Portfolio

Welcome to my **Project Management Portfolio Website**, built to showcase my professional experience, project case studies, skills, and personal background as I work through the **Merit America Google Project Management Program**.

This site highlights both my **real-world leadership experience** and my journey into formal project management training.  
It is a live, responsive, modern portfolio designed with **HTML**, **CSS**, and **JavaScript**.

This portfolio contains:
